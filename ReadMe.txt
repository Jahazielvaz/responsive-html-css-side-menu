This project is meant to show you exactly how to make a responsive sidebar nav in pure CSS HTML and a little JS. No frameworks, no libraries, just pure languages.

Here's a list of the items you'll learn.
-How to make an svg hamburger.
-How to make a side nav bar and have the content not overlap, as well as having the content slide to the right.
-Advanced targeting.
-How to add functionality with JS by using the getElementById.
-Best styling practices. 
-How to get inline content to expand or contracts simply by adding and removing, or increasing it's container width.
